# Kreativ Haus 77

Site para agência de marketing focada em resultados reais. Não entregamos postzinho bonito, entregamos aumento de faturamento e mais vendas.

## Tecnologias

- Next.js 15
- TypeScript
- Tailwind CSS v4
- React 19

## Desenvolvimento

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Build de produção
npm run build

# Iniciar servidor de produção
npm start
```

## Features

- **Iceberg Visualization**: Representação visual mostrando que marketing é mais do que posts e tráfego pago
- **Design Sober**: Cores escuras com destaques em pink e roxo
- **Responsive**: Layout adaptável para todos os dispositivos
- **Performance**: Site estático otimizado com Next.js
